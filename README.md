## 🌟 Tentang Edugenzi
Edugenzi adalah platform website profil untuk tempat kursus teknologi yang dirancang dengan metode belajar sambil praktik. Edugenzi berfokus pada pengembangan keterampilan digital anak dan remaja melalui proyek nyata yang membangun kreativitas dan kemampuan *problem solving*.

## 🛠 Tech Stack
Project ini dibangun menggunakan teknologi web modern untuk menjamin performa yang cepat dan kode yang *maintainable*:
- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Library**: [React](https://reactjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)

## ✨ Highlight Fitur UI/UX
- **Desain Modern & Playful**: Menggunakan palet warna yang cerah dan *friendly* (Cream, Green, Yellow, Purple) yang cocok untuk target audiens anak dan remaja.
- **Glassmorphism & Shadows**: Elemen UI menggunakan efek transparan yang elegan dan *soft shadows* untuk memberikan kesan premium.
- **Responsif**: Layout yang rapi dan optimal di berbagai ukuran perangkat, dari *smartphone* hingga *desktop*.
- **Komponen Modular**: Struktur komponen yang rapi seperti `SectionHeading`, `Card`, dan `Container` untuk konsistensi desain.

## 🚀 Getting Started

Ikuti langkah-langkah berikut untuk menjalankan project ini secara lokal:

### 1. Install Dependencies
Pastikan Anda sudah berada di dalam direktori project, lalu jalankan:
```bash
npm install
# atau
yarn install
# atau
pnpm install
```

### 2. Jalankan Development Server
```bash
npm run dev
# atau
yarn dev
# atau
pnpm dev
```

Buka [http://localhost:3000](http://localhost:3000) di browser Anda untuk melihat hasilnya secara langsung. Project ini mendukung *Hot Module Replacement* (HMR), sehingga perubahan kode akan langsung terlihat.

## 📂 Struktur Folder
- `src/app/` - Routing utama aplikasi (Next.js App Router).
- `src/components/sections/` - Komponen besar yang membentuk bagian-bagian halaman utama (Hero, ValueProps, FAQ, dll).
- `src/components/ui/` - Komponen dasar dan *reusable* (Button, Card, dll).
- `src/content/` - Data statis yang menggerakkan konten dinamis di website (Data Program, FAQ, Testimoni).
